# Портфолио фотографа — Идея

## Суть
Сайт-портфолио для фотографа с разноплановыми съёмками (театр, семейные, портреты, мероприятия). Одновременно работает как витрина для клиентов и как кейс веб-разработчика.

## Целевая аудитория
- Частные клиенты (семейные съёмки, портреты)
- Компании и организации (танцевальные студии, театры, мероприятия)

## Жанры съёмок
- Театр
- Семейные
- Портреты
- Мероприятия / Events
- Mix (другие форматы)

## Функциональность
- Галереи фотографий по категориям
- Отзывы клиентов
- Анимации и эффекты
- Форма связи / заявки

## Стек
- React (основа)
- Vite (сборка)
- Three.js / GSAP / Framer Motion (анимации — по необходимости)

## Нерешённые вопросы
- Дизайн: тональность, цветовая схема, язык сайта
- Галереи: категории с фильтрами или отдельные страницы
- Отзывы: статичные или с возможностью оставить через сайт
- Нужна ли CMS / админка для фотографа
- Хостинг, домен, SEO
- Intro-анимация (в процессе обсуждения)
